# PBM Pre-Mobile Social + Notifications Update

Bu paket mobil uygulama/Capacitor kodu icermez.

Icerik:

- X benzeri PBM Social timeline
- Buyutulebilen Social gorselleri
- Web bildirim merkezi ve bildirim zili
- Admin postundan sonra Supabase web bildirimi
- Resend email bildirimi
- Cloudflare custom domain aciklamasi

## GitHub'a yukleme

ZIP'i ac. Icindeki klasorleri ve dosyalari mevcut PBM repo kokune yukle.
GitHub mevcut dosyalarin ustune yazacaktir.

## Supabase

Supabase SQL Editor'de bir kere calistir:

```text
supabase/social_notifications_20260606.sql
```

## Cloudflare variables

Email bildirimi icin:

```text
RESEND_API_KEY
PBM_EMAIL_FROM
PBM_SITE_URL
```

Bu paket mevcut dashboard, chart, AI analiz ve PBM tasarim motorunu degistirmez.
